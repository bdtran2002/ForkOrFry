# Security Policy

ForkOrFry is an early-beta hobby project. We do not run production services.

## Supported surface

- extension UI and local storage
- local game state
- build and packaging scripts

## What to report

Please report:

- data loss
- local privilege or storage issues
- extension security problems
- accidental exposure of secrets in the repo or build output

Please do not report routine gameplay bugs here.

## How to report

Use GitHub Security Advisories if available, or email the maintainer directly if a private path exists. If neither is available, open a minimal issue and avoid including sensitive details.

## Expectations

- response time may be slow
- fixes may land on the main branch or a beta release line
- this project is not a formal security-supported product
