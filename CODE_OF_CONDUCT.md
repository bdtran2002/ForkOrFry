# Code of Conduct

We want this project to stay practical and respectful.

## Expected behavior

- be civil and direct
- focus on the work, not the person
- assume good intent
- keep feedback specific and actionable

## Unacceptable behavior

- harassment or abuse
- discriminatory language or behavior
- personal attacks
- repeated bad-faith disruption

## Enforcement

This is a small hobby project in early beta. Maintainers may remove comments, close issues, or block access if behavior makes collaboration harder.

If something feels off, open a private security note or contact a maintainer directly.
